react 组件/library 项目模板

## Usage

```bash
npx degit ximing/react-library-tpl  my-project
```

## Feature

- 支持 jest 测试
- 使用 gatsby 创建官网
- src 和 example 使用 typescript
- 支持发布到 github pages
- 使用 github workflows 自动发布官网
- 使用 rollup 进行构建
- eslint/typecheck 支持
- git commit 时进行代码检测
